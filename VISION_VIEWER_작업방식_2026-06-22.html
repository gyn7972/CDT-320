요청 1 — 동작 버튼 누르는 동안 LIVE 유지 코드완료

  • 뷰어 패널이 CameraViewBase.AttachSource(VisionViewerSource)로 연결 → 화면의 네이티브 툴바 Grab/Live/Stop이 그대로 동작.
  • 패널에 별도 LIVE / STOP / GRAB 버튼 추가, 핸들 생성 시 자동 LIVE 시작 → 액션 버튼을 눌러도 영상이 계속 흐름.
  • 동작 테스트 버튼(EXPOSE/MATCH 등)은 여전히 모션 미구동(약속 유지).
+ Ui\Controls\VisionViewerPanel.cs (Live/Stop/Grab 바 + 자동 Live)

요청 2 — GRAB 시 결과 데이터 오버레이(Vision PC처럼) 코드완료

  • VisionViewerSource에 FrameMeta 이벤트 추가 — 프레임마다 메타(스케일/판정/결과라인/마크)를 통지.
  • 뷰어 패널이 메타를 받아 SetVerdict / SetResultLines / SetOverlay / MmPerPixel로 오버레이 표시 → 판정·검출 마크·측정 mm가 영상 위에 그려짐.
  • GRAB 버튼 = 명령 채널로 EXPOSE 전송(촬상·검사 트리거) → 결과 프레임이 뷰어로 들어와 오버레이 갱신.
~ Equipment\Vision\VisionViewerSource.cs (FrameMeta 이벤트)

요청 3 — Front Side + Rear Side 동시 표시 코드완료

  • SideVisionViewerControl 신설 — 위=FRONT SIDE(TopSide 5205), 아래=REAR SIDE(BottomSide 5206)를 반반 동시 표시.

  • 실측 이미지 4000×700은 추후 분할 예정 — 지금은 전체를 그대로 표시(셋업 편의).
